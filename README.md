### An easy plugin to build month two dimensional heatmaps

#### Features
- Mobile friendly/ responsive
- Data in `json` format
- Uses [rollup.js](https://github.com/rollup/rollup)
- Clean code

#### Data structure

```
{
  "Avocado": { "Jan":2, "Feb":1, "Mar":0, ...},
}
```

#### How
`npm install`   
`npm start`

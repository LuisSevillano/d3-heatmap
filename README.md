### An easy plugin to build two dimensional heatmaps

#### Features
- Mobile friendly/ responsive
- Data in `json` format
- Uses [rollup.js](https://github.com/rollup/rollup)
- Clean code   
- [Demo](https://www.luissevillano.net/d3-heatmap/index.html)

![thumbnail](https://www.luissevillano.net/img/heatmap-big.jpg)
#### Data structure

```json
"Avocado": { "Jan":2, "Feb":1, "Mar":0},

```

#### How
`npm install`   
`npm start`
